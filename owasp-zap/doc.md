# 🛡️ OWASP ZAP (Zed Attack Proxy)

## 📌 What is OWASP ZAP?

**OWASP ZAP (Zed Attack Proxy)** is a free, open-source security testing tool used to find vulnerabilities in web applications.

👉 In simple words:
**It acts like a hacker (ethically) to test how secure your app is.**

---

## 🚀 What can you do with OWASP ZAP?

### 🔍 1. Scan your application

* Automatically find vulnerabilities like:

  * SQL Injection
  * Cross-Site Scripting (XSS)
  * Security misconfigurations

---

### 🕷️ 2. Discover all URLs (Spider)

* Crawl your app and find:

  * Hidden pages
  * APIs
  * Endpoints

---

### ⚡ 3. Attack your app (Active Scan)

* Send malicious requests to test:

  * Input validation
  * Backend security

---

### 🔐 4. Test login & authentication

* Check:

  * Login forms
  * Sessions
  * Tokens

---

### 🧪 5. Intercept & modify requests

* Works as a proxy to:

  * Change headers
  * Modify parameters
  * Replay requests

---

### 📡 6. Test APIs

* Scan REST / Swagger APIs directly

---

### 📊 7. Generate security reports

* Get vulnerabilities with:

  * Severity (High / Medium / Low)
  * Fix suggestions

---

### 🔁 8. Automate in DevOps

* Run scans in:

  * Docker
  * CI/CD pipelines (GitHub Actions, Jenkins, Azure DevOps)

---

👉 **OWASP ZAP = Tool to find security issues in your app before hackers do**

---

## 🧠 Use Cases

* Secure web applications before production
* Perform penetration testing
* Integrate security into DevOps (DevSecOps)
* Test APIs and microservices
