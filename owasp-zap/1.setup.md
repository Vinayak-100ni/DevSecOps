1 install docker on server 
```bash
docker network create zap-net
docker pull bkimminich/juice-shop 
docker run --rm -itd --name juice-shop -p 3000:3000 bkimminich/juice-shop
docker  network connect zap-net juice-shop 
docker run -u zap -d --name zap --network zap-net -p 8080:8080 ghcr.io/zaproxy/zaproxy:stable zap-webswing.sh
```

2. Verify Connectivity
```bash
docker exec -it "container-id" curl http://juice-shop:3000
```
3. open the zap using url   --- > ip:8080/zap
   
4.Spider (Discover Endpoints)
```
Steps:  Click Quick Start

Enter target: http://ip:3000

Click Attack

This will: Crawl your app & Find all URLs/endpoints
```
5. Active Scan (Find Vulnerabilities)

After spider completes:

Steps:

```
Go to Sites tab
Right-click your app URL
Click Attack → Active Scan
  This will test: SQL Injection, XSS, Security issues
```
6. API Testing
If your app has APIs:

Import OpenAPI:
```
Go to Import → OpenAPI
Add Swagger URL or file
👉 ZAP will scan APIs directly
```
